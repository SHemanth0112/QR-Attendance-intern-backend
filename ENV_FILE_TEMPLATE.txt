Copy this content to create your .env file in the backend folder:

PORT=5000
MONGODB_URI=mongodb+srv://hemanthkumarsurisetti_db_user:YOUR_ACTUAL_PASSWORD@qr.ba5pjfg.mongodb.net/qr-attendance?retryWrites=true&w=majority
JWT_SECRET=qr_attendance_secret_key_2024_change_in_production
JWT_EXPIRE=7d
NODE_ENV=development


INSTRUCTIONS:
============

1. Create a new file named ".env" in the backend folder (not .env.txt, just .env)

2. Copy the content above into the .env file

3. Replace "YOUR_ACTUAL_PASSWORD" with your actual MongoDB Atlas database password

4. If your password has special characters, URL encode them:
   @ becomes %40
   # becomes %23
   $ becomes %24
   % becomes %25
   : becomes %3A
   / becomes %2F

5. Save the file

6. Make sure your IP is whitelisted in MongoDB Atlas:
   - Go to https://cloud.mongodb.com
   - Click "Network Access" 
   - Add your IP or use 0.0.0.0/0 for development

7. Start the server with: npm run dev


Example with password "MyPass@123":
